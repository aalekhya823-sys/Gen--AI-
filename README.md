# Gen--AI
# 🏏 IPL RAG Chatbot

## 📌 Overview

This project is a **RAG-based chatbot** that answers IPL questions using **unstructured PDF data** and **Groq API**.

## 🚀 Features

* Extracts text from IPL PDF
* Uses embeddings for search
* Generates answers using Groq LLM

## 🛠️ Tech Stack

Python, LangChain, Groq API, FAISS/ChromaDB, Streamlit

## ▶️ Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 🔑 Setup

Set your Groq API key:

```bash
set GROQ_API_KEY=your_api_key
```

## 💬 Example

Ask: *"Who won IPL 2023?"*
